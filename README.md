# flappyPenguin
Flappy-bird clone using SDL2 

## Dependencies
  `make`
  `SDL2`
  `SDL2_image`

![Screenshot](https://github.com/GwakaMolie/flappyPenguin/blob/master/screenshot.png)


## Clone
  `git clonehttps://github.com/GwakaMolie/flappyPenguin.git && cd flappyPenguin`
  
## Build and Complie
  `make`

## Run 
  `./game`
